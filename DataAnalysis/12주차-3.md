### 1. 시간복잡도

- 복잡도(Complexity) : 알고리즘의 성능을 나타내는 척도
- 시간 복잡도 : 특정 크기의 입력에 대해 알고리즘 수행 시간 분석
- 공간 복잡도 : 특정 크기의 입력에 대해 알고리즘의 메모리 사용량 분석

<빅오 표기법 (Big-O Notation)>
